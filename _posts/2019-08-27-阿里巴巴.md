4. 概率题：给定一条线段，任意裁成三段，能组成三角形的概率；
-- 概率是一个小三角形除以一个大三角形，0.25；许久不做概率题，首先要把抽象问题具体化，数字化，符号化，比如不失一般性，令线段总长为1，其中两条边为x和y，则第三边为1-x-y，一个三角形是0 < x < 0.5，0 < y < 0.5和x + y > 0.5，另一个三角形是0 < x < 1，0 < y < 1和0 < x + y < 1，最终得到答案0.25；
